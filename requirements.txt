# ============================================================================
# SLDPC: runtime dependencies
#
# Versions below mirror the verified paper-reproduction environment
# (Python 3.9, NVIDIA GPU, CUDA 11.7) exactly. The same combination has
# been used to produce all numbers reported in the paper.
#
# Default install command:
#
#     pip install -r requirements.txt
#
# This pulls the GPU build of PyTorch from PyTorch's CUDA 11.7 index.
# The CUDA 11.7 wheels are forward-compatible with newer NVIDIA drivers,
# so this combination works on machines with CUDA 11.7 / 11.8 / 12.x
# drivers without changes.
#
# CPU-only / no NVIDIA GPU
# ------------------------
# If you have no NVIDIA GPU (e.g. on a Mac, or a CPU-only Linux box), the
# ``--index-url`` line below will fail to resolve a wheel. Replace the
# three pinned ``torch*`` lines with their plain CPU equivalents, e.g.::
#
#     torch==2.0.1
#     torchvision==0.15.2
#
# and remove the ``--index-url`` line. Training will run on CPU but be
# significantly slower.
# ============================================================================
